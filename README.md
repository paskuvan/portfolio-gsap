Modern **UX/UI Portfolio** built with **React, GSAP, and Tailwind CSS** to showcase projects, animations, and design work in an interactive way.

This portfolio focuses on **smooth motion design, performance, and accessibility** while presenting professional work in **UX/UI and frontend development**.

---

## ✨ Features

- 🎨 Modern UX/UI portfolio layout  
- ⚡ Smooth animations using **GSAP**  
- 📱 Fully responsive design  
- 🎯 Optimized performance with **Vite**  
- 🧩 Modular React components  
- 🌙 Clean and minimal interface  
- 🖱️ Scroll-based and interactive animations  

---

## 🛠️ Tech Stack

- **React**
- **GSAP (GreenSock Animation Platform)**
- **Tailwind CSS**
- **Framer Motion**
- **Vite**
